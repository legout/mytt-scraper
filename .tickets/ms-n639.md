# Deliver login/logout flow and protected route enforcement

Build authenticated access with login form, SSE auth status updates, logout handling, and auth guard on protected routes.

## Design

PRD US-1 + 4.3; Spec 2.1(auth),2.2(session),3.1; Plan P1.1-P1.5

## Acceptance Criteria

- [ ] /login renders validated credentials form with loading feedback
- [ ] Login POST runs blocking auth via asyncio.to_thread()
- [ ] Successful auth persists session user and redirects
- [ ] Invalid auth shows user-visible error state via SSE
- [ ] /logout clears session and protected routes redirect if unauthenticated
